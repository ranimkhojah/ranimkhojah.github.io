NLP-based tool to optimize test case prioritization

The programmed tool uses Natural Language Processing (NLP) to process test cases specification documents and prioritize the corresponding test cases according to a diversity-based test prioritization. 2019

C.A.R.L.O.S (Cyber Physical Systems and Systems of Systems)

Carlos is a scaled variant of a real car that is programmed to perform self-driving tasks including maneuvering, detecting traffic signs using object- and color-detection and handling various intersection scenarios. These tasks are done through the interaction between 4 microservices that communicate over 2 main OD4 sessions, with the input provided by 2 ultrasonic sensors, 2 infrared sensors and 1 RPI camera.

MBTI personality type predictor (Machine Learning)

The Machine Learning model used is trained on Myers-Briggs Personality Type dataset (MBTI) and performs Natural Language Processing (NLP) to predict users' personality types by analysing their tweets, facebook posts and/ or written thoughts. The MBTI predictor also performs validation on the trained data to maximize the precision of the prediction.

WALL-E (Arduino)

WALL-E is built using an Arduino-car kit. It was programmed in Arduino/C++ to perform basic car tasks automatically and remotely, however the main task that WALL-E performs is parallel parking in 3 different scenarios using a Raspberry Pi, Raspi Camera, 1 gyroscope and 3 Ultrasonic sensors.
